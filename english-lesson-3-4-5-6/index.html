Siri が何回やっても、「I bought that」、「I called cats」 としか聞きとってくれない。。

2020-05-30 英会話 英語ガク習塾

なかなか気が乗らないアウトプットですが、何とか書き始めることができました。 書き始めたら割と一気にできるので、書き始めるコツを調べて今後記事にしたいと思っています。

今回の記事では、Lesson 3 から Lesson 6 までの内容をやっていきます。 Lesson 3 から Lesson 6 は、Lesson 1 と Lesson 2 で習った母音の復習を兼ねて母音の練習をやっていきます。

母音について

母音でとても大切なことは、英語の母音のほとんどの口の形は、日本語よりも、かなり大きい！！ 例えば、/ ʌ / という発音記号ですが、色々な記事で口の形を小さくあけると書いてあることがありますが、英語の母音の中で一番小さいだけで日本語の「ア」という音を出した時の口の形の大きさと比べると大きいです。 つまり、口を小さくするという意識はありません。

母音の長さ

英語では、母音の長さと単語の意味とは関係がありません。 日本語で、「doki」と「do-ki」では意味が全く違いますが、英語では短くても長くても変わりません。 大切なのは、子音と母音によって生み出される発声が意味を持たらすということです。

大きさを表わす表

ここに表に書いてある図形は、口の形をイメージしています

/ ʌ /, / e /, / æ /, / ɑː /

この4つはまとめて覚えていきます。 それぞれのの口の大きさの違いと、口の形の図形イメージが分かる表があるのでそれを使って覚えましょう。

1. / ʌ / の発音

英語の文字

「u」の文字

舌位置

後舌

口形と発声

日本語の「ア」と同じ位の口の大きさで軽くあひる口をして、喉の奥が閉まるような感じで「ア」と発声します。

2. / e / の発音

英語の文字

「e」の文字

舌位置

前舌で、下の歯につくような感じ

口形と発声

口を両手の小指で広げて、上の歯が「キラーン」と見えるような感じで「エ」と発声

3. / æ / の発音

英語の文字

「a」の文字

舌位置

前舌で、舌の歯にピッタリつく。舌が一瞬下に落ちる

口形と発声

人差し指と中指を縦にして口に入れて歯にあたる位、もしくは、薬指と小指を縦にして歯に当たらない位口を開け、高速で「ェア」と発声します 。口を動かさないことがポイントです。

4. / ɑː / の発音

英語の文字

「o」の文字

舌位置

後舌で、喉の奥に落ちるような感じ

口形と発声

人差し指と中指を縦にして口に入れて歯にあたらない位開けて「オ」と発声します。 「ア」の音素を含んだ「オ」と捉えて良いですし、「オ」の音素を含んだ「ア」と捉えてても良いです。

/ æ / と / ɑː /

bat と cat の練習をする前に

1. / b / の発音

まずは 日本語の「ブ」と発音してみてください。日本語は唇が離れた時に「ブ」と発声します。 英語は、くっついてる唇が勢いよく離れる時に「ブ」と発声します。ドラムの音をイメージしてください。

破裂音

2. / k / の発音

喉の奥で息を止めて、いっきに吐き出すような感じ 摩擦音

o u a が c の後に続く時は、 / k / の音になり、 i e が c の後に続く時は、 / s / になります。

3. / t / の発音

上顎に舌をつけて離した時に音がなる。ドラムのステアのような音

I bought bats と I caought cats で練習

bought と bot の発音と caught と cot の発音は一緒

英語を発音する上でルールがあります。通称 フォニックスルールというのがあります。単語を満ただけで発音ができるというルール。種類が多いのと例外が多いので、いきなりルールを覚えようとするのは大変ですが、今回は一つだけ覚えましょう。 それは、「aught」 と 「ought」を含む単語は文字の部分の発音が、 / ɑː / の発音になるというルールです。
