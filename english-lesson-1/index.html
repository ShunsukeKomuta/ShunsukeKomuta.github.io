英語ガク習塾 Lesson 1 〜謝るのにもヒト苦労！？〜

2020-04-29 英会話 英語ガク習塾

いざ記事を書こうとすると億劫で、ゴールデンウィークで時間があったのにも関わらず 今回の Lesson 動画を見てから一週間も経ってしまいました(汗)
しかし、気合を入れて今回の Lesson 動画で学んだことをアウトプットしていきたいと思います。

英語ガク習塾の進め方のおさらい

手鏡を持って口の形をしっかり認識し手発音すること

英語ガク習熟のレッスンの進め方の特徴的なポイントとして、手鏡を必ず持つということがあります。これは、口の形が違ければ発音が異なるからです。

レッスン中は声を大きく発音する

日本語は母音が5つしかありません。この5つしかない母音を組み立てる能力が日本人にはあり、口を抑えた発音でも理解することができます。そのため日本語は子音が弱いのです。
一方でアメリカ英語は強い子音が多く小さく発音するとより伝わりづらいです。日本語の1.5倍くらいの大きさで発声することを心掛けてください。

今日覚えたことを誰かに何も見ないで教えること

Lesson 0 で繰り返し強調していますが、その日勉強したことを必ず誰かに何も見ずに教える。

I’m sorry

この Lesson では、「I’m sorry」の発音を練習していきます。

I’m / aɪm /

I’m の部分はかなり聞き取りづらいです。英語は日本語と違って構造を重視する言語で、主語が基本的には存在します。
そのため、主語をつけるという癖をつけてください。
しかし、発音はかなり弱く発音されます。実際には、「アイム」とは聞こえず、「ム」とだけ聞こえたり、ほとんど聞こえないこともあります。

ちなみに、文章では、I am と丁寧に書かれることはありますが、会話ではほぼ 100% I’m になります。

そして、発音に入る前に発音の重要なポイントについて触れておきます。
まず一つ目は、「口の形」です。これは日本人と外国の人の口の形を見比べればすぐに分かると思います。
二つ目は、「舌の位置です。」発音しながら確かめてもらえば分かりますが、日本語の母音は舌の位置がほとんど変わりません。日本語母音の位置は中舌です。英語では2つの発音を除いて、全ての前舌もしくは後舌で発音を行います。

1. / aɪ / の発音

「I’m」の アの発音は、下の歯に舌をギュットくっつけて発音します。発音記号は、「a」です。続けて、イの発音ですが、重要な点はアとイが別々に発音している訳ではないということです。アイという一つの発音になり、舌の歯に舌をつけた状態で「アイ」と発音します。イの音記号は、「ɪ」です。「ɪ」は日本語のエほど口を上下に開かずイほど口を左右に開かず前舌でイと発音する感じです。個人的にはエと聞こえます。
また、似た発音記号に「i」がありますが、これは口を横にめいいっぱい広げて口幅はかなりせまくします。その状態で、舌を上の歯につけてイーと発音します。

2. / m / の発音

「m」の発音は、上唇と下唇を閉じた状態から、唇が離れる時に「ムッ」と発音します。

sorry / sɑri /

1. / s / の発音

日本語のハヒフヘホは強い音ですが、英語の / h / は弱く消えやすい感じです。

「s」の発音は、口を閉じ少し歯が見える状態で、息を漏らすような感じで発音します。「sɑː」の発音をそのまま発音すると、少し馬鹿っぽく聞こえますが、発生後少し音程を下げてください。英語は発音だけでなく音程も大事です。リーの発音は、「ri」です。

2. / ɑ / の発音

「ɑː」の発音は、口を大きく開け後舌でアーと発音します。指二本が縦に入る位大きく口を開けて練習すると良いそうです。
慣れてくるとそこまで大きく口を開けなくても「ɑː」の発音ができるようになるそうですが、 私は「ɑː」の発音をする時に舌が前に残ってしまうので、常に指を入れて練習した方が良さそうです。

3. / r / の発音

「r」の発音は、日本語で「アイウエオ」を発音する中舌の位置から、舌をお皿のような形にして、両脇が上の奥歯にあたるような感じで発音します。
中舌でウとオの間のような発音をしながらこの形にしてまた元に戻してを繰り返し練習をしていきます。

4. / i / の発音

「i」の発音は、「i」でも「I」でもどっちでも良いです。これは、アクセントのないところは力が抜けてどっちもよくなるからです。そんな適当な！！
