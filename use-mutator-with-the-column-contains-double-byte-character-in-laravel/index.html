Laravel - テーブルに日本語を含むカラムに対して、ミューテーターを使用したい時

結論から言うと、モデルに対して以下の2つを行う。 ただし、日本語名をそのまま使用するという訳にはいかない。

  • 新しくアクセッサーを作成する
  • 属性名をモデルの $appends プロパティに追加する。(アクセッサーはキャメルケースで書かれているが、属性名はスネークケースで記入する)
<?php

class Sale extends Eloquent {

    protected $appends = ['price'];

    public function getPriceAttribute()
    {
        return $this->getAttribute('料金') * 1.08;
    }

}

言うまでもないけれども、日本語のターブル名やカラム名は使わなければこんなことをしなくてすむ。
