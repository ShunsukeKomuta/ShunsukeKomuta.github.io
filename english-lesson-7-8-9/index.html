「You can't do that」の発音は今までのレッスンができていれば簡単！！

2020-06-06 英会話 英語ガク習塾

今回の記事では、Lesson 7 から Lesson 9 までの内容をやっていきます。



レッスンのフレーズ

今回のレッスンで扱うフレーズは「You can’t do that」です。 意味は、「なんでそんなことするの。 = やめてよ。」となります。

can と can’t の違いが分からない?

can は / kæn / と発音し、can’t は / kænt / と発音します。しかしアメリカ英語には、ストップTというルールがあり、会話中では、どちらも発音は、/ kæn / と発音します。 では、どうやって can か can’t かを判断するかというとストレス(強勢)で判断します。 / kæn / の後の動詞にストレスが来ていれば、can になり、/ kæn / にストレスが来ていれば can’t ということになります。

アメリカ英語には、単語の語尾に来る子音、 特に t k p d g b は 止まって音が聞こえなくなる という特性があります。 下記のようにセットで覚えると覚えやすいです。

  • t (摩擦音) と d (t の有声音)
  • k (摩擦音) と g (k の有声音)
  • p (破裂音) と b (p の有声音)

ちなみに、n と t は舌の位置と使い方が似ています。n と t は音の出し方が近いため、t で音を止めるとほとんど n に聞こえます。 なので、can も can’t も n で終わっているように聞こえます。そのため、肯定文と否定文でストレスの置く場所を変えて判断します。

that について

that と it の違い

感情的にポジティブ、もしくはフラットな場合、「this」か「it」を使い、ネガティブな気持ちが入る場合は、「that」を使います。

例えば、洋服屋で買いたいも洋服がある場合は、 「I’ll take it」と言い、 要らないものは、「I don’t need that.」と言います。

物理的な距離感は一緒。何が違うかというと、「心の距離」です。 that を使うことによって、心の距離が遠い = 嫌なことを表わします。

舌位置

口形と発声

口は日本語の「ウ」に近い感じで、少し口の形は丸くする感じ。 日本語の「ウ」は「ウ」と認識できる範囲が広いですが、外国語では「ウ」に似た音で異なる意味を持つ言語があります。 喉の奥で閉まったような感じの音になります。
