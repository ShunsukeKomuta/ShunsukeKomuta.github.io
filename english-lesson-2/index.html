「that」の発音が難し過ぎる!

2020-05-24 英会話 英語ガク習塾

今日も気合入れて、ガク習塾で学んだことをアウトプットしていきたいと思います。

この Lesson1 では、「How’s that?」の発声を練習していきます。 「How’s that?」の意味は「どういうこと?」です。 「How’s」は、「How is」と書かれていても、省略系で発音します。これは、「I’m」と一緒ですね！

that は「あれ」「それ」という意味ですが、前の文章全体を指すこともあります。今回学ぶ、この使い方の that です。 あなたが前に言ったことは、どういうこと? という意味になります。

How’s / haʊz /

まずは、How’s について学んでいきます。/ h /, / aʊ /, / z / の 3つに分解できます。

1. / h / の発音

口形と発声

日本語のハヒフヘホは強い音ですが、英語の / h / は弱く消えやすい感じです。

2. / aʊ /

注意して欲しいのが、 / a / + / ʊ / ではなく、これが一つの音という点です。このように2つの母音が切れ目なく発音される母音のことを複合母音(二重母音)と言います。

舌位置

後舌で、舌が喉の奥に来るような感じです

口形と発声

「ウ」の口で、「オ」と発声します。「ウ」と「オ」の中間音になります。前舌の位置から後舌の位置に移動しますが、後舌の位置に来た時には、喉の奥をギュっと閉めるような感じです。

3. / z /

口形と発声

/ s / を有声音にします。摩擦音を追加すような感じです。日本語で「ズ」と発声すると「ウ」の音が強いですが、/ z / では「ウ」の音はないので注意してください。

that / ðæt /

次に、that について学んでいきます。/ ð /, / æ /, / t / の 3つに分解できます。

1. / ð /

舌位置

前舌で、上の歯と下の歯をちょっと開けて、その間に舌を入れます。この時、舌は歯で挟まないのですが、難しいという人は挟んでも大丈夫です。

口形と発声

上からも下からも空気を出し、空気を出している状態から舌を引きます。そして、引いた直後に母音を出します。

2. / æ /

舌位置

前舌で、下の歯に下をつけます

口形と発声

親指を縦にして入る位口を開けて、高速で、「ェア」と発声します。「ア」と「エ」の中間音です

3. / t /

舌位置

上顎のところにつけます

口形と発声

ドラムのスネアのような音です。舌を口の上から離して、「トゥッ」と発音します。ここで一つ覚えておいてほしいのが、英語のストップTというルールです。 /t/ で終わる場合は、舌を上顎のところにくっつけた後に舌を離さずに音が出さないようにします。

例えば、「meet」や、「seat」などがそうですね。また、助動詞の否定系の「don’t」、「shouldn’t」、「wouldn’t」、「couldn’t」などは全部も全部そうです。

I’m sorry. How’s that? を続けて

この文章で良いたいのは、「I’m sorry」の sorry と 「How’s that」の that です。 ここだけが文章で聞こえれば伝わります。なので、文章でアクセントが必要になります。文章では伝わりにくいのですが sorry と that にアクセントつけて、さらにテンポをつけます。言いたいところにアクセントとテンポをつける、出すことが英語では重要になるので、意識していくようにします。
